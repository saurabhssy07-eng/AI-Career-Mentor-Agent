# AI Career Mentor Agent

## Microsoft Agents League Hackathon 2026 Submission

AI Career Mentor Agent is a multi-agent reasoning system built using Microsoft Azure AI Foundry. The solution helps students, fresh graduates, and career switchers make informed career decisions through resume analysis, skill-gap assessment, personalized learning roadmaps, certification recommendations, interview preparation, and career guidance.

---

## Problem Statement

Many students and early-career professionals struggle with:

* Understanding their current skill level
* Identifying gaps for target roles
* Choosing the right certifications
* Creating realistic learning roadmaps
* Preparing for technical interviews
* Deciding between multiple career paths

Career guidance is often fragmented across websites, videos, forums, and online courses.

---

## Solution

AI Career Mentor Agent uses a multi-agent architecture where specialized reasoning agents collaborate to analyze a user's profile and generate a structured career development plan.

The system provides:

* Profile Analysis
* Skill Gap Assessment
* Career Roadmap Planning
* Certification Recommendations
* Project Suggestions
* Interview Preparation
* Final Career Recommendations

---

## Multi-Agent Architecture

### 1. Profile Analysis Agent (PAA)

Responsibilities:

* Analyze user profile
* Identify strengths
* Identify weaknesses
* Extract relevant experience
* Generate structured profile summary

Output:

* User profile assessment

---

### 2. Skill Gap Analysis Agent (SGA)

Responsibilities:

* Compare current skills against target role
* Identify missing competencies
* Prioritize learning requirements

Output:

* Skill gap report

---

### 3. Roadmap Planning Agent (RPA)

Responsibilities:

* Create personalized learning roadmap
* Define milestones
* Organize learning phases

Output:

* Structured learning plan

---

### 4. Certification & Project Recommendation Agent (CPRA)

Responsibilities:

* Recommend industry certifications
* Suggest portfolio projects
* Align learning with career goals

Output:

* Certification roadmap
* Project recommendations

---

### 5. Final Aggregation & Recommendation Agent (FARA)

Responsibilities:

* Aggregate outputs from all agents
* Generate final career strategy
* Present actionable recommendations

Output:

* Comprehensive career guidance report

---
